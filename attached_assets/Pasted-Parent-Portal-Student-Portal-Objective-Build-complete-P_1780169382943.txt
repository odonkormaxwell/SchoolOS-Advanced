Parent Portal & Student Portal
Objective

Build complete Parent and Student Portals for Edulex.

The Parent Portal and Student Portal should be clean, modern, mobile-friendly, and easy to use.

These portals should become one of Edulex's strongest selling points by giving parents and students access to real-time academic and financial information.

Role permissions and data scoping rules must be strictly enforced.

Parents should only see their linked children.

Students should only see their own records.

No administrative features should be accessible from these portals.

PORTAL ARCHITECTURE

Create separate portals:

Parent Portal

Route:

/parent

Student Portal

Route:

/student

Users should automatically be redirected to the correct portal after login.

Example:

Parent Login

↓

Parent Dashboard

Student Login

↓

DATA SECURITY

Critical.

Parent Scope

Parents must only access:

Their linked children.

Required checks:

parent_id
student_id
school_id

Block access to other students.

Student Scope

Students must only access:

Their own record.

Required checks:

student_user_id
student_id
school_id

Block access to all other students.

Direct URL Protection

If a parent or student manually enters:

/students

/settings

/payments

/reports

/admin

Result:

Access Restricted

Message:

You do not have permission to access this page.
