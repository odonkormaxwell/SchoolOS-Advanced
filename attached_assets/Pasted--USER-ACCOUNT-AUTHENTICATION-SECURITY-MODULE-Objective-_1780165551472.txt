# USER ACCOUNT, AUTHENTICATION & SECURITY MODULE

## Objective

Improve the Edulex user account system so users can securely log in, log out, edit their profiles, change passwords, reset passwords, and optionally enable Two-Factor Authentication.

The current system has issues where users cannot properly log out or edit their profile. This must be fixed immediately.

---

# 1. AUTHENTICATION PAGES

Create or fix the following pages:

## Login Page

Route:

/login

Requirements:

* Email field
* Password field
* Remember Me checkbox
* Forgot Password link
* Login button
* Clear error messages
* Loading state
* Redirect user based on role after login

Role redirects:

* Super Admin → Super Admin Dashboard
* Administrator → School Dashboard
* Teacher → Teacher Dashboard
* Accountant → Finance Dashboard
* Admissions Officer → Admissions Dashboard
* Parent → Parent Portal
* Student → Student Portal

---

## Register Page

Route:

/register

This should be for new school registration only.

Fields:

* School Name
* Contact Person Name
* Email
* Phone Number
* Password
* Confirm Password

After registration:

* Create school account
* Create first Administrator user
* Redirect to onboarding/setup wizard

Do not allow students, parents, or staff to self-register in Version 1.

They should be created by the school administrator.

# 5. TWO-FACTOR AUTHENTICATION

Create:

Account → Security → Two-Factor Authentication

Support 2FA for:

* Super Admin
* School Administrator
* Accountant

Optional for:

* Teacher
* Admissions Officer
* Parent
* Student

## 2FA Methods

Version 1:

* Email OTP

Future Ready:

* SMS OTP
* Authenticator App
* WhatsApp OTP

## 2FA Workflow

When enabled:

1. User enters email and password.
2. System verifies credentials.
3. System sends OTP to email.
4. User enters OTP.
5. System verifies OTP.
6. User is logged in.

OTP Rules:

* 6-digit code
* Expires after 10 minutes
* Maximum 5 attempts
* Resend OTP after 60 seconds
* Log failed attempts
