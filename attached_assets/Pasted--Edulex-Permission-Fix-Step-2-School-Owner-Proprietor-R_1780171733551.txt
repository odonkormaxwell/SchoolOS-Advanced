# Edulex Permission Fix – Step 2: School Owner (Proprietor) Role Only

## Objective

Fix and enforce the School Owner (Proprietor) role throughout the entire system.

The School Owner is the owner of the school business.

This role focuses on:

* School performance
* Revenue
* Billing
* Staff oversight
* School branding
* System settings
* User management
* Backups
* Compliance

The School Owner is NOT responsible for day-to-day academic operations.

Those responsibilities belong to the Headteacher.

This separation must be strictly enforced.

# Students Module

School Owner should have limited student access.

Can:

* View students
* Search students
* View student profiles
* View enrollment statistics

Cannot:

* Enter scores
* Approve results
* Mark attendance
* Manage assessments

The School Owner should not be managing academic records.

---

# Staff Module

School Owner can:

* View staff
* Create staff accounts
* Edit staff accounts
* Archive staff accounts
* Reset staff passwords
* Activate users
* Deactivate users

Can view:

* Teacher assignments
* User activity
* Last login

Cannot:

* Assign academic results
* Enter attendance

---

# User Management

School Owner is the primary user manager within the school.

Can:

* Create users
* Edit users
* Suspend users
* Activate users
* Reset passwords
* Assign roles

Roles available:

* Headteacher
* Teacher
* Accountant
* Admissions Officer
* Parent
* Student

Cannot:

* Create Super Admins
* Modify platform-level roles

---

# Finance Module

School Owner should have full financial oversight.

Can:

* View fee structures
* Create fee structures
* Edit fee structures
* Archive fee structures
* View invoices
* View payments
* View balances
* View receipts
* View finance reports

Can approve:

* Fee structure changes
* Offline payment approvals
* Payment reversals

Cannot:

* Delete payments
* Delete invoices
* Delete receipts

Use:

* Reverse
* Void
* Archive

instead of delete.

---

# School Owner Restrictions

School Owner must NOT:

* Enter assessment scores
* Mark attendance
* Approve results
* Publish results
* Reopen results
* Lock results
* Assign subject scores

These responsibilities belong to the Headteacher and Teachers.
